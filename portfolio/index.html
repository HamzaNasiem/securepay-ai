AMD Developer Hackathon: ACT II — Unicorn Track

#AI #Fintech #Security #AMD #Docker

SecurePay AI is a hardware-accelerated fraud prevention engine and disposable payment tokenization platform. The system generates merchant-locked, spend-capped virtual card tokens to eliminate credit card data leaks, using local and serverless Explainable AI models to analyze risk vectors and provide plain-language security reasoning in real-time.

🛡️ Zero-Leak Cryptographic Vault: Real card data is encrypted using AES-256-GCM and stored locally, completely isolated from merchant networks.
🧠 Explainable AI Risk Engine: DeepSeek V4 Pro processes metadata (location/device/token age) and provides detailed explanation logs.
🛑 Subscription Kill Switch: Fully-synchronized Redis status updates enable users to instantly revoke token access with one click.
⚡ GPU Telemetry Integration: Visualizes processing latencies, token consumption, and cost savings directly on the dashboard.
